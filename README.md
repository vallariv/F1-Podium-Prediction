# 🏎️ F1 Podium Prediction using Machine Learning

## 📌 Project Overview

This project predicts whether a Formula 1 driver will finish on the podium (Top 3) using Machine Learning techniques. The model is trained on historical Formula 1 race data after performing data preprocessing, feature engineering, model training, and evaluation.

The objective is to analyze race-related features and build a predictive model that can estimate a driver's chances of achieving a podium finish.



## 🚀 Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Multiple Machine Learning Models
- Model Performance Evaluation
- Feature Importance Analysis
- Podium Prediction Function


## 🤖 Machine Learning Models Used

- Random Forest Classifier
- XGBoost Classifier
